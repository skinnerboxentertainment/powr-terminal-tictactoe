# Research Solicitation: ANSI/ASCII Terminal Aesthetic Reference

**Project:** WarGames Tic Tac Toe — Cold War CRT terminal game
**Context:** Browser-based tic-tac-toe with a retro military-computer visual identity. Current render is a CRT monitor housing with dark screen, pale icy-blue grid, and status readouts. Need to push further into the ANSI/terminal/TUI aesthetic.

## Research brief

Research the following aesthetic markers and return a curated reference document. For each marker, provide: (1) what defines it visually, (2) concrete examples (artists, tools, artworks), (3) techniques that could translate to a live PixiJS canvas rendering.

### Core markers

- **ANSI art** — The 80s/90s BBS scene, TheDraw, ACiD, iCE. Block characters, dithering, 16-color palette constraints, "ice" colors, ansi-mation layers. Specifically: how title screens, logos, and menus were composed from shaded block characters.
- **ASCII art** — Monochrome line art, density gradients (`.` `:` `-` `=` `+` `*` `#` `%` `@`), silhouette construction, large-scale portraits and logos from characters alone.
- **Terminal/TUI design** — Layout grammar of teletext, VT100, IBM 3270, early BBS interfaces. Status bars, bordered panels, prompt cursors, tab-stops, form fields. How these layouts create hierarchy without graphics.
- **CRT/phosphor aesthetic** — Scanlines, bloom, ghosting, monitor curvature, green/amber/white phosphor colors, low resolution, analog signal degradation. Not just a filter — the feel of light generated by electron beams hitting phosphor.
- **BBS "door" game splash screens** — TradeWars 2002, LORD, Legend of the Red Dragon, Operation: Overkill II — their title screens, login prompts, and status HUDs. The "dark sci-fi utilitarian" layout language.
- **3D illusion in 2D terminal art** — Techniques used in ANSI to fake depth: beveled borders, drop shadows, extrusion lines, faux-lighting via character dithering, isometric block art.

### Specific references to find

1. ANSI art title screens that say "TIC TAC TOE" or similar simple game names
2. TheDraw screenshot collections showing logo construction (shadow, outline, fill, highlight)
3. BBS door game HUD layouts — the way stats and readouts are arranged
4. Terminal status panels with labeled values (like the Wargames missile-command readouts)
5. The specific cyan-on-navy ANSI "ice" color scheme used in dark BBS themes
6. Examples of animated ANSI (ansimation) — flickering cursors, loading bars, rotating prompts

### What we need from the research

- A vocabulary of terminal/ANSI visual elements
- Technical patterns we can adapt (e.g., morphological character dilation for outlines, dithering patterns for gradient, ordered noise for CRT grain)
- Reference images or descriptions we can use as direct style targets

### Deliverable

A markdown document with sections per marker, each containing:
- Description of the visual technique
- 2-3 concrete examples
- Notes on how it could map to a PixiJS v8 canvas rendering (Graphics, Text, blend modes, filters)
